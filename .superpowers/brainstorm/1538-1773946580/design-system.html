Key changes from current site

  • Light mode default — warm ivory (#FAFAF8) replaces dark navy
  • Warm neutrals — every neutral tinted toward stone/cream, no pure black or white
  • Accents punctuate, don't dominate — cyan for CTAs and links only, purple for differentiation moments
  • Real typography — distinctive display font + readable body font, no system defaults
  • No gradient text, no glass cards, no blur orbs
